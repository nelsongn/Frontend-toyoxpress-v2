# ToyoXpress Dashboard v2 

Interfaz administrativa de alto rendimiento construida con **Next.js** y **Tailwind CSS**. Diseñada para ofrecer visibilidad en tiempo real sobre el estado de sincronización y ventas.

## Mejoras del Proyecto respecto a la version anterior
- **Dashboard de Sincronización:** Interfaz interactiva que muestra el progreso global de carga de productos y manejo de errores.
- **UI Profesional:** Implementación de componentes modulares utilizando **Shadcn UI** y **Radix UI**, priorizando la usabilidad sobre el diseño artístico.
- **Gestión de Estado:** Manejo eficiente de flujos de autenticación y permisos de usuario mediante `useAuthStore`.

## Tecnologías
- **Framework:** Next.js 14+ (App Router)
- **Estilos:** Tailwind CSS
- **Componentes:** Shadcn UI / Lucide React
- **Validación:** Integración con API REST personalizada.

## Instalación
1. Clonar el repositorio.
2. `npm install`
3. Configurar `.env` con la URL del Backend.
4. `npm run dev`
